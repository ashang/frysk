Frysk

Overview

The Frysk project aims to produce a Execution Analysis Tool that lets linux application developers and system administrators take a deeper look at their running applications. We aim to both focus on and exploit the capability of a pure open-source Linux Environments.

Command Line Build

Check out frysk:
cvs -d :pserver:anoncvs@sources.redhat.com:/cvs/frysk co frysk
Build the configuration files:
( cd frysk && ./autogen.sh )
Create a separate build directory, and build in there:
( mkdir build && cd build && ../frysk/configure && make)

Eclipse Build

For eclipse, check out the cvs module frysk-eclipse.
