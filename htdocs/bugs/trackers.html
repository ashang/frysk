frysk Tracker Bugs

All frysk tasks (bugs, features, changes), hang off of one or more of these trackers:
2236: documentation (including web pages)
This includes both integrated and web documentation.
1600: the build and test environment
This includes problems building and testing frysk, problems porting frysk to a specific system.
2235: build problems
2234: Testing problems
2104: Fedora Core 5
2004: Red Hat Enterprise Linux 4
1976: Ubuntu
1975: Fedora Core 4
2229: Base System
The frysk base system consisting of:
1580: event loop
The event-loop manages low-level asynchronous events such as signals, timers, and ready file descriptors, feeding them into to frysk's core.
1582: proc model
The process model implements the abstraction of a host's process / thread tables.
1599: lang model
The language model implements an abstraction of programming language concepts such as frame, method, and variable.
1601: utilities
The utilities include ftrace and fstack.
2230: Java-GNOME User Interface:
1632: monitor view
The monitor view visualizes frysk's process model.
1633: source view
The source view visualizes frysk's language model.
1602: upstream
Tracks problems with upstream code such as java-gnome and gcc. For each problem there is typically a corresponding test case.
1496: kernel
Tracks changes needed from the Linux Kernel.
1875: features
Tracker for feature requests.
