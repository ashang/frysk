Below is a work-in-progress snapshot of a process view. A user is in the process of adding an observer to the exec process. In addition to being logged, a user can associate actions to an observer. Examples of observers and actions include:

  • monitor TestRunner processes
    prompt to kill when one exceeds more than 5 minutes of cpu
  • monitor gnome-terminal, and its children
    prompt for the source browser when the program eclipse is started.
